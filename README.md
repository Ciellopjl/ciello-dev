## 👨‍💻 Sobre o Projeto

Este é o meu portfólio profissional e painel administrativo de alta performance. Desenvolvido com o que há de mais moderno no ecossistema **Next.js 16**, o projeto não é apenas uma vitrine visual premium, mas uma aplicação full-stack completa. Conta com gerenciamento de conteúdo dinâmico (CMS próprio), segurança avançada e análise de métricas nativa.

> **Objetivo Principal:** Demonstrar forte competência técnica em arquitetura full-stack, segurança web de ponta e excelência em experiência do usuário (UX).

---

## ✨ Funcionalidades em Destaque

### 🛡️ Segurança & Autenticação de Nível Enterprise
- **Autenticação Segura (NextAuth.js):** Controle rigoroso de acesso administrativo garantindo que apenas usuários na whitelist tenham permissão.
- **Firewall Reativo (Security Monitor):** Sistema inteligente que registra, analisa e bloqueia automaticamente tentativas de acesso indevidas e maliciosas.
- **Edge Middlewares:** Rotas blindadas a nível de borda, impedindo acessos sem autorização muito antes de chegarem à lógica da aplicação.

### ⚙️ Headless CMS & Analytics (Admin Dashboard)
- **Gestão Fluida de Projetos:** CRUD completo para inserção, edição detalhada e publicação instantânea dos projetos em vitrine.
- **Drag-and-Drop Intuitivo:** Reordenação visual do portfólio e componentes de UI interativos desenvolvidos do zero.
- **Advanced Analytics Dashboard:** Coleta de pageviews, conversões de cliques, dispositivos e gráficos dinâmicos com `Recharts`.
- **Inteligência Artificial (Groq/Llama):** Geração e otimização automatizada de tags e features de projetos impulsionada por IA direto no painel.

### 🎨 Frontend Moderno & UX Premium
- **Tailwind CSS 4:** Arquitetura de design system escalável, responsivo e de extrema leveza estrutural.
- **Framer Motion:** Micro-interações, efeitos de *tilt* 3D e transições imersivas.
- **Gestão na Nuvem (Cloudinary):** Otimização massiva e entrega ultrarrápida de imagens enviadas.

## 🛠️ Stack Tecnológica de Alta Performance

A arquitetura foi planejada pensando em escalabilidade de longo prazo e manutenibilidade limpa.

| Camada | Tecnologia | Propósito no Projeto |
| :--- | :--- | :--- |
| **Framework** | **Next.js 16 (App Router)** | Renderização mista (SSR/SSG), Server Actions e rotas de API nativas |
| **Linguagem** | **TypeScript** | Forte tipagem e prevenção de erros em tempo de compilação |
| **Estilização** | **Tailwind CSS 4** | Design responsivo veloz e focado em utilitários |
| **Banco de Dados** | **PostgreSQL (Neon)** | Armazenamento relacional distribuído, robusto e flexível |
| **ORM** | **Prisma** | Controle fluido do esquema do banco e migrations *typesafe* |
| **Autenticação** | **NextAuth.js** | Integração multi-provider ágil e segura com JWT Sessions |
